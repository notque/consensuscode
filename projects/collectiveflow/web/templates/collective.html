How to Participate

1. View Proposals: Browse current proposals above or on the main page. No authentication required - all information is transparent.

2. Submit Proposals: Use the CollectiveFlow CLI to create new proposals: ./collectiveflow proposal create

3. Provide Consultation: Share your input on active consultations: ./collectiveflow consensus input [proposal-id]

4. Build Consensus: Work together to address concerns and find agreement. Every voice matters equally.
